# @erwanjugand/prettier-config

Prettier preset.

## Installation

```
npm install -D @erwanjugand/prettier-config
yarn add -D @erwanjugand/prettier-config
pnpm add -D @erwanjugand/prettier-config
bun add -D @erwanjugand/prettier-config
```

## Usage

Create an `prettier.config.js` file and import the preset.

```js
import config from "@erwanjugand/prettier-config";

export default config;
```
